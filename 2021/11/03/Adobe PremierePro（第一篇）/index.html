Adobe PremierePro（第一篇）

发表于2021-11-03|更新于2021-11-08
|阅读量:

关于Adobe PremierePro的应用以及一些剪辑知识，也是为了温故知新，也是为了巩固基础，所以再次学习一遍。
本博客参考bilibili课堂的Pr收费课程，本篇从电子相册为案从零开始学习剪辑软件Adobe PremierePro。
Point 1：
Adobe、Adobe PremierePro 与剪辑。Adobe PremierePro，简称Pr，是从属于Adobe系列的剪辑工具，除了Pr外，其实还有不少好用的强力剪辑软件，但Pr受到欢迎的最重要原因，大概还是因为它便于同其他Adobe系列的工具形成协同效益，从而实现更多功能、大大提高效率。
Point 2：
关于素材和成片。一个视频的创作流程一般是：策划、拍摄、剪辑。而剪辑则是将拍摄环节得到的素材加工为成片的过程。
Ponit 3：
新建项目。文件（F）+新建（N）+项目（P） 或者 Ctrl + Alt + N 。一般而言新建项目的属性只修改名称和文件所在位置即可。
Point 4：
调整工作区块。一个项目在工作中会用到多个面板，所以依照个人口味合理调整工作区块是很实用的技能。调整工作区块有两种方式：一是拖动调整；二是右键选择某一区块浮动。如果调整得不尽人意，想要恢复的话，通过窗口（W）+工作区（W）即重置为已保存的布局，或者Alt + Shift + 0亦可。
Point 5：
导入素材。要用到项目面板。素材的导入有两种方式：（1）双击项目面板的空白处（2）选中素材拖动到项目面板。在导入素材之后，我们还可以对素材做一个整理，在项目面板下方有“新建素材箱”功能，相当于创建了一个项目内的素材文件夹。
关于项目面板，还有如下功能：放大至全屏（全部面板都可实现），选中某一面板，按“~”（英文模式下）；返回上层；搜索；锁定；视图模式……
Point 6：
新建序列。要用到时间轴面板。在项目面板中应用“新建项”功能，选择“序列”选项，对于新建的序列属性，我们可以选择如下配置：“编辑模式”、“时基：25帧/秒”（时基与素材帧率匹配）、“帧大小：1920 x 1080”、“像素方宽：方形像素”、“场：无场（逐行扫描）”、名称自拟。
调整时间轴，可以通过滚轮和滑块调整时间轴具体位置，也可以在英文模式下通过“\”键将时间轴间距缩放到合适大小。
素材导入时间轴时要注意：图像素材要导入到“V”轨道（Video），而且图像素材有层次顺序，上方轨道的素材会遮盖住下方轨道的素材；音频素材要导入到“A”轨道（Audio）；当一个完整的视频被导入时，会同时占用两个轨道。
我们可以用音频剪辑混合器调整音量。
在时间轴面板上，还有诸多功能工具：对齐工具能够将素材自动吸附对齐；链接选择项能够自动同时选中视频素材中的图像和音频两个部分；添加标记能够在总序列的某一时刻添加备忘标记。
这里重点介绍剃刀工具和选择工具，这两个功能简单但是非常重要：剃刀工具（C）用以截选素材；选择工具（V）用以拖动、编辑素材。shift+可以让剃刀工具在某一时间点将所有素材截断。
Point 7：
效果控件面板：对素材主体添加效果：有运动、不透明度和时间重映射三种效果。
效果面板：对视频整体加工：可供选择的有预设、音频效果、音频过渡、视频效果、视频过渡。
使用Ctrl + D 可以快速添加默认转场效果。在效果面板中，右击效果即可设置为默认转场，而在编辑-首选项-时间轴中可以编辑默认效果的具体属性。单击某一效果，可以具体编辑单个效果。
Point 8：
当你对一个素材进行效果加工后，想要批处理该如何做：右击处理好的素材，选择复制（Ctrl+C），框选待处理的素材，右击，粘贴属性（Ctrl+Alt+V）。
Point 9：
最后是关于光晕素材，光晕素材在使用时需要加工处理：在效果空间面板中将不透明度的混合模式改为滤色，否则光晕素材会全部盖住其他图像素材。
Point 10：
导出视频。文件（F）+导出（E）+媒体（M）或者Ctrl + M 。格式选择H.264，预设使用高比特率，输出名称自拟。
